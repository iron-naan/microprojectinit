# Novel Combination Risk Visualiser

This prototype lets you type in multiple risks and visualise a simple interaction between them. The output is generated locally using mock data, but the logic could be replaced with OpenAI API calls.

## Usage

Open `index.html` in a modern web browser. Enter at least two risks separated by commas and click **Visualize**. A brief text description and a small node-link diagram will appear.

The code is intentionally lightweight and meant for experimentation only.
